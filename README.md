# EcomRank - Sistem Pendukung Keputusan (SPK) E-commerce

EcomRank adalah aplikasi berbasis web yang dirancang untuk membantu menentukan platform e-commerce terbaik menggunakan metode **Simple Additive Weighting (SAW)**. Proyek ini dibangun dengan teknologi modern untuk memberikan hasil perankingan yang akurat dan responsif.

## 🚀 Fitur Utama

- **Manajemen Kriteria**: Kelola kriteria penilaian (Benefit/Cost) beserta bobotnya.
- **Manajemen Alternatif**: Kelola daftar platform e-commerce (Shopee, Tokopedia, dll).
- **Matriks Penilaian**: Input nilai setiap alternatif dengan presisi hingga 4 desimal.
- **Visual Rangking**: Hasil peringkat otomatis dengan badge khusus untuk Juara 1, 2, dan 3.
- **Database Terintegrasi**: Menggunakan PostgreSQL (Supabase/pgAdmin).
- **Export Excel**

## 🛠️ Tech Stack

- **Frontend**: [Next.js](https://nextjs.org/) (App Router), TypeScript, Tailwind CSS.
- **Backend/Database**: [Supabase](https://supabase.com/) (PostgreSQL).
- **Icons & UI**: Lucide React & Framer Motion.
